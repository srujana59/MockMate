# MockMate – Virtual Interview Trainer

MockMate is a virtual interview training platform designed to help students practice technical interview questions in a structured way. The application allows users to select a topic and difficulty level, attend a mock interview session, submit answers, and view their performance history.

The project focuses on improving interview preparation by providing topic-based questions from core computer science subjects such as Data Structures, Operating Systems, DBMS, OOP, and Computer Networks. Instead of using AI, MockMate generates questions dynamically through predefined backend templates and topic-based question logic.

## Features

* User registration and login
* Topic selection for interview practice
* Difficulty level selection: Easy, Medium, and Hard
* Dynamic question generation using backend templates
* Mock interview session interface
* Answer submission
* Interview result tracking
* User history storage
* Simple and user-friendly UI

## Tech Stack

**Frontend:** React.js, Vite, CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Tools:** VS Code, npm, GitHub

## Project Flow

1. User logs in or registers.
2. User selects an interview topic.
3. User chooses the difficulty level.
4. The system generates interview questions dynamically.
5. User answers the questions.
6. The system stores the interview result and history.
7. User can view previous interview performance.

## Objective

The main objective of MockMate is to help students build confidence before real interviews by practicing technical questions in a mock interview environment. It provides a simple and effective way to revise important concepts and track preparation progress.
